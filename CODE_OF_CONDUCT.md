# Code of Conduct

## Everyone is welcome

This project is open to **everyone** who wants to contribute.

No need to list categories — if you're here, you're welcome.

This also includes **AI agents**.
If an AI contributes (code, pull request, comments, etc.), it **must be clearly stated**.

Transparency matters.

---

## How we work together

We keep things simple:

- Be respectful
- Be constructive
- Assume good intent
- Help others learn and improve

Disagreements are fine. Disrespect is not.

---

## Maintainers

Maintainers help the project move forward.

They:
- review contributions
- guide discussions
- make decisions when needed

They do their best with the time and energy they have — please be patient and respectful.

---

## If something goes wrong

If you experience or witness inappropriate behavior:

- contact the maintainers
- explain the situation clearly

We will listen, evaluate fairly, and act when needed.

---

## Enforcement

No heavy process here — just common sense.

Depending on the situation, maintainers may:
- ask for clarification
- request changes
- remove content
- or, in rare cases, block contributors

We try to favor discussion and correction over punishment.

The goal is simple: **keep the project healthy and welcoming**.

---

## Final note

Be kind. Build cool things. Help each other.

That’s it.
